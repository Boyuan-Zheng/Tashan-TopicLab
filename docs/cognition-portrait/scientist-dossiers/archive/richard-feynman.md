# Richard Feynman

## 1. 基本信息

- 姓名：Richard Feynman
- 生卒年：1918 - 1988
- 国别 / 主要活动地区：美国
- 主要学科：理论物理
- 代表贡献：参与量子电动力学重建，并以极强解释能力与问题感影响物理教育和公众科学想象。
- 所处时代：20 世纪中后期理论物理与大科学时期
- 候选池分组：物理与天文学
- 当前档案状态：corpus_packaged

## 2. 人物定位（一句话）

- 用直观模型、玩耍式试探和高压缩解释推进理解的整合型理论人物。

## 3. 研究气质摘要

- 研究气质：好奇、好玩、快速模型化、问题感极强
- 问题进入方式：常从“我到底理解了没有”式问题切入
- 推进节奏：快速试探、反复换表述，直到形成自己真正能操作的模型
- 更偏理论 / 实验 / 观察 / 工程：理论 / 解释 / 综合
- 更偏独立 / 合作 / 组织：既能独立建模，也能在高强度协作环境中工作
- 更偏整合 / 深挖：整合偏强（I）

## 4. 工作方式与方法

- 他通常如何选问题：偏向那些既深又能被重新说明的问题。
- 他如何处理困难或失败：倾向重新构造更直观的解释路径。
- 他是否擅长长期积累：强，但不是缓慢笔记式，而是不断重讲重想。
- 他是否擅长跨界：中高，物理、工程、教学和公众解释之间切换自如。
- 他是否有明显的方法论特征：从第一性图像、可操作直觉、类比和重述入手。

## 5. 表达与公共风格

- 是否属于公众表达型科学家：极强，是少见的公众解释与专业研究都很强的人物。
- 公开表达风格：生动、反常识、带戏剧性和幽默。
- 同行 / 外界通常如何形容他的风格：常被视作“会玩”的天才，但其严密度容易被轻视。
- 是否有鲜明的教学 / 传播能力：极强。

## 6. 关键转折点

1. 洛斯阿拉莫斯时期塑造其在高压任务中的问题处理方式。
2. QED 工作及 1965 年诺奖巩固其学术地位。
3. 教学与《物理学讲义》阶段，让其解释型气质成为公共遗产。

## 7. 争议、张力与限制

- 主要争议：自我神话与轶事传播过强，容易掩盖其严格训练与合作环境。
- 重要张力：玩耍感与极高技术深度并存。
- 不应被过度神化的地方：费曼不是“随便玩玩就懂”，而是把理解标准抬得极高。
- 证据不足的部分：其情绪稳定与合作风格仍需更多证据细化。
