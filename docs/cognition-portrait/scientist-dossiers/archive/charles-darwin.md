# Charles Darwin

## 1. 基本信息

- 姓名：Charles Darwin
- 生卒年：1809 - 1882
- 国别 / 主要活动地区：英国
- 主要学科：生物学、自然史
- 代表贡献：以自然选择理论重塑生物演化理解，并通过巨大 correspondence network 组织证据。
- 所处时代：19 世纪自然史与专业科学转型期
- 候选池分组：生物、医学与生命科学
- 当前档案状态：corpus_packaged

## 2. 人物定位（一句话）

- 极其耐心的观察-整合型研究者，通过长期积累和广泛通信把分散事实织成理论。

## 3. 研究气质摘要

- 研究气质：谨慎、耐心、证据导向、网络化整合强
- 问题进入方式：从观察材料和异常现象切入，再慢慢织出总体解释
- 推进节奏：长周期收集，延迟发表，高度审慎
- 更偏理论 / 实验 / 观察 / 工程：观察 / 综合 / 理论
- 更偏独立 / 合作 / 组织：高度依赖通信网络与样本网络，但核心判断多由本人沉淀
- 更偏整合 / 深挖：整合偏强（I）

## 4. 工作方式与方法

- 他通常如何选问题：偏向能解释广泛自然现象的统一问题，但不急于定论。
- 他如何处理困难或失败：倾向继续搜证、继续通信、继续比较，而不是快速下结论。
- 他是否擅长长期积累：极强，靠笔记、标本、实验与通信共同维持。
- 他是否擅长跨界：中高，能把地质、育种、地理分布等线索连起来。
- 他是否有明显的方法论特征：观察、比较、分类、通信验证、延迟发表。

## 5. 表达与公共风格

- 是否属于公众表达型科学家：不是表演型人物，但其书写能力和论证结构极强。
- 公开表达风格：谨慎、层层推进、不断预设反对意见。
- 同行 / 外界通常如何形容他的风格：常被视作耐心与证据积累的典范。
- 是否有鲜明的教学 / 传播能力：极强，通过《物种起源》完成高影响力科学传播。

## 6. 关键转折点

1. “贝格尔号”航行让其获得决定性的观察材料。
2. 1837-39 年笔记期形成自然选择核心框架，但长期未公开。
3. 华莱士来信促使其在 1858-59 年加速公开理论。

## 7. 争议、张力与限制

- 主要争议：其理论后来被社会达尔文主义等外推误用，需和本人科学主张区分。
- 重要张力：谨慎求证与理论爆炸性影响并存。
- 不应被过度神化的地方：达尔文不是“瞬间想通”，而是长期证据编织。
- 证据不足的部分：其健康问题与家庭、通信网络对工作节奏的影响需继续细化。

## 8. 适合作为哪类画像锚点

- 主锚点候选：观察-整合型、长期积累型、谨慎发布型人物
- 备选锚点候选：使命驱动型、证据织网型
- 更适合支持的维度：整合、尽责、低冲动、观察型研究
- 不太适合支持的维度：高速迭代、强舞台型表达
