# Marie Curie

## 1. 基本信息

- 姓名：Marie Curie
- 生卒年：1867 - 1934
- 国别 / 主要活动地区：波兰出生，主要活动于法国
- 主要学科：物理、化学
- 代表贡献：开创性推进放射性研究，发现并分离钋与镭，并在医学成像应用上有现实贡献。
- 所处时代：20 世纪初实验科学高强度实验室时期
- 候选池分组：物理与天文学
- 当前档案状态：corpus_packaged

## 2. 人物定位（一句话）

- 高度克制、极端耐久、实验与使命感结合很强的深挖型人物。

## 3. 研究气质摘要

- 研究气质：坚韧、克制、纪律性强、实验耐力极高
- 问题进入方式：从测量异常和材料性质切入，持续打磨实验路径
- 推进节奏：长期高强度耐力推进
- 更偏理论 / 实验 / 观察 / 工程：实验 / 深挖
- 更偏独立 / 合作 / 组织：早期与皮埃尔协作紧密，后期能独立扛起整个实验室方向
- 更偏整合 / 深挖：深挖偏强（D）

## 4. 工作方式与方法

- 他通常如何选问题：偏向基础而艰苦、需要长期实验坚持的问题。
- 他如何处理困难或失败：更倾向硬撑与继续推进，而非频繁换题。
- 他是否擅长长期积累：极强，靠长时间重复实验与测量。
- 他是否擅长跨界：中，能把基础研究和医疗应用连接，但核心仍是实验深挖。
- 他是否有明显的方法论特征：高强度实验、测量、纯化、重复、忍耐。

## 5. 表达与公共风格

- 是否属于公众表达型科学家：个人风格克制，但社会象征性极强。
- 公开表达风格：不以花哨表达见长，更像肃静而坚定的实验室人物。
- 同行 / 外界通常如何形容他的风格：常被描述为坚忍、专注、工作伦理极强。
- 是否有鲜明的教学 / 传播能力：公众影响大，但核心不是话语魅力，而是工作本身。

## 6. 关键转折点

1. 赴巴黎求学并进入索邦体系，是其科学路径的关键起点。
2. 与皮埃尔的合作及放射性研究奠定其核心成就。
3. 皮埃尔去世后继续主导实验室，并在一战中推进移动 X 光设备。

## 7. 争议、张力与限制

- 主要争议：公众常将其神圣化为纯粹牺牲者，容易掩盖其方法与判断力。
- 重要张力：高度私人克制与极高公共象征性并存。
- 不应被过度神化的地方：不能只讲“苦”和“奉献”，要看到其精准而顽强的实验能力。
- 证据不足的部分：其具体实验室管理风格仍可继续补证。

- 不太适合支持的维度：高外向、轻松玩耍型解释者
