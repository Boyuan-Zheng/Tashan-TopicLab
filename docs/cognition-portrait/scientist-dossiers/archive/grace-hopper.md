# Grace Hopper

## 1. 基本信息

- 姓名：Grace Hopper
- 生卒年：1906 - 1992
- 国别 / 主要活动地区：美国
- 主要学科：计算机科学
- 代表贡献：推动编译器、程序语言标准化和计算教育传播，是从机器到人类可读语言的重要桥梁人物。
- 所处时代：早期计算机工程与软件语言形成期
- 候选池分组：工程、信息、系统与跨界建构
- 当前档案状态：corpus_packaged

- 他是否擅长长期积累：高，体现为系统落地和语言生态。
- 他是否擅长跨界：中高，从数学、海军、工业软件到教育传播。

- 是否属于公众表达型科学家：强讲解型工程人物。
- 公开表达风格：犀利、幽默、直给。
- 同行 / 外界通常如何形容他的风格：常被视作计算机语言与软件工程先驱。
- 是否有鲜明的教学 / 传播能力：很强，尤其在工程教育层面。

## 6. 关键转折点

1. 加入 Harvard Mark I 项目，进入早期计算核心现场。
2. 在 Eckert-Mauchly / UNIVAC 路线推动编译器和 Flow-Matic。
3. 后期在海军和业界推动语言标准化与年轻工程师培养。

## 7. 争议、张力与限制

- 主要争议：围绕“她是否独自发明 COBOL”等叙事需要避免神化。
- 重要张力：她既是工程先驱，也是制度与训练体系建设者。
- 不应被过度神化的地方：不能把她简化成几个流行梗；她真正重要的是降低计算复杂度。
- 证据不足的部分：更多一手技术文档还可继续系统补齐。
