# Alan Turing

## 1. 基本信息

- 姓名：Alan Turing
- 生卒年：1912 - 1954
- 国别 / 主要活动地区：英国
- 主要学科：计算理论、逻辑
- 代表贡献：奠定可计算性理论，推动密码分析与早期计算机思想，深刻影响 AI 与认知讨论。
- 所处时代：20 世纪中期计算奠基期
- 候选池分组：数学、逻辑与计算基础
- 当前档案状态：corpus_packaged

## 2. 人物定位（一句话）

- 把极高抽象模型与可实现机器问题连在一起的深挖型桥接者。

## 3. 研究气质摘要

- 研究气质：概念敏锐、独立、安静、带强烈问题执念
- 问题进入方式：从“什么可被形式化 / 机械化”这样的根问题切入
- 推进节奏：少数关键论文和工程阶段构成高强度跃迁
- 更偏理论 / 实验 / 观察 / 工程：理论 / 工程桥接 / 深挖
- 更偏独立 / 合作 / 组织：能在战时团队中工作，但核心洞见高度个人化
- 更偏整合 / 深挖：深挖偏强（D）

## 4. 工作方式与方法

- 他通常如何选问题：偏向定义边界、定义可能性的根问题。
- 他如何处理困难或失败：通过改写问题形式来获得清晰答案。
- 他是否擅长长期积累：高，但输出集中。
- 他是否擅长跨界：高，逻辑、密码、机器、心智问题之间有天然联通。
- 他是否有明显的方法论特征：形式系统、边界判定、原型机器思维。

## 5. 表达与公共风格

- 是否属于公众表达型科学家：生前并非公众人物，后世象征性很强。

- 是否有鲜明的教学 / 传播能力：后世影响远大于生前声量。

## 6. 关键转折点

1. 1936 年关于可计算数的论文定义其根本地位。
2. 布莱切利园密码分析将理论能力接到极端现实任务。
3. 战后计算机设计与机器智能问题拓展其影响范围。

## 7. 争议、张力与限制

- 主要争议：国家迫害和个人悲剧是其历史叙事不可回避的部分。
- 重要张力：高度抽象与极端现实任务并存。
- 不应被过度神化的地方：不能只把他写成“悲剧天才”，要看到其问题定义能力。
- 证据不足的部分：其情绪稳定和外向性判断应保持谨慎。
