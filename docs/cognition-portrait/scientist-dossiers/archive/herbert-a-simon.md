# Herbert A. Simon

## 1. 基本信息

- 姓名：Herbert A. Simon
- 生卒年：1916 - 2001
- 国别 / 主要活动地区：美国
- 主要学科：认知科学、系统科学
- 代表贡献：以有限理性、组织决策、人工智能和设计科学重塑多学科对“人如何判断”的理解。
- 所处时代：20 世纪跨学科社会科学与 AI 奠基期
- 候选池分组：工程、信息、系统与跨界建构
- 当前档案状态：corpus_packaged

## 2. 人物定位（一句话）

- 把决策、认知、组织和计算连成一体的系统整合者。

## 3. 研究气质摘要

- 研究气质：框架化、跨学科、持续建模、乐于解释
- 问题进入方式：从复杂现实决策问题切入，抽出可操作模型
- 推进节奏：长期稳定输出，跨领域并行推进
- 更偏理论 / 实验 / 观察 / 工程：综合 / 模型 / 组织
- 更偏独立 / 合作 / 组织：强协作、强学派建设
- 更偏整合 / 深挖：整合偏强（I）

## 4. 工作方式与方法

- 他通常如何选问题：偏向跨学科共通机制，如判断、搜索、设计、组织。
- 他如何处理困难或失败：通过简化模型和认知架构重写问题。
- 他是否擅长长期积累：高，体现在一整套跨学科框架。
- 他是否擅长跨界：极强。
- 他是否有明显的方法论特征：模型化、认知约束分析、组织与计算结合。

## 5. 表达与公共风格

- 是否属于公众表达型科学家：不是大众明星，但在学术世界中解释力很强。
- 公开表达风格：清楚、教学感强、概念命名能力强。
- 同行 / 外界通常如何形容他的风格：常被视作“真正跨学科”的罕见人物。
- 是否有鲜明的教学 / 传播能力：学术传播强。

## 6. 关键转折点

1. 《Administrative Behavior》确立其组织决策问题意识。
2. 与 Newell 等合作推动 AI 与认知建模。
3. 获得 Turing Award 与诺奖，确认其跨领域影响。

## 7. 争议、张力与限制

- 主要争议：其框架有时被批评过于模型化，但这恰是其优势与限制同源。
- 重要张力：现实复杂度与模型简化之间始终存在张力。
- 不应被过度神化的地方：不能只记“有限理性”四个字，要看到其更广的人工科学项目。
- 证据不足的部分：大众层资料较少，更多需依赖学术回忆与机构档案。
