# Linus Pauling

## 1. 基本信息

- 姓名：Linus Pauling
- 生卒年：1901 - 1994
- 国别 / 主要活动地区：美国
- 主要学科：化学、分子结构
- 代表贡献：以化学键理论、分子结构研究和后期公共反核行动展示了少见的科学-公共双重影响力。
- 所处时代：20 世纪化学与分子生物学扩展期
- 候选池分组：化学、材料、地球与环境
- 当前档案状态：corpus_packaged

## 2. 人物定位（一句话）

- 能把理论化学、结构问题和公共科学角色连接起来的高开放整合者。

## 3. 研究气质摘要

- 研究气质：想象力强、框架感强、敢于跨出学科边界
- 问题进入方式：从结构关系和分子形态切入，再向更广领域推进
- 推进节奏：持续高产，后期向公共议题大幅外扩
- 更偏理论 / 实验 / 观察 / 工程：理论 / 综合
- 更偏独立 / 合作 / 组织：能在实验和理论共同体中发挥中心作用
- 更偏整合 / 深挖：整合偏强（I）

## 4. 工作方式与方法

- 他通常如何选问题：偏向能用统一原则解释复杂结构的问题。
- 他如何处理困难或失败：通过更大胆的结构假设和概念工具推进。
- 他是否擅长长期积累：高，体现在跨化学、生物和公共议题的持续输出。
- 他是否擅长跨界：高。
- 他是否有明显的方法论特征：结构推断、量子化学思维、概念统一。

## 5. 表达与公共风格

- 是否属于公众表达型科学家：后期是强公共知识分子。
- 公开表达风格：直接、带行动号召。
- 同行 / 外界通常如何形容他的风格：既被视为 20 世纪伟大化学家，也因后期部分健康主张引发争议。
- 是否有鲜明的教学 / 传播能力：强。

## 6. 关键转折点

1. 早期化学键研究确立其科学核心地位。
2. 《化学键的本质》时期形成分子结构整合能力。
3. 反核试验行动与和平奖使其公共角色显著放大。

## 7. 争议、张力与限制

- 主要争议：后期维生素 C 等主张削弱了部分科学共同体对其判断的信任。
- 重要张力：卓越科学判断与后期部分争议性公共立场并存。
- 不应被过度神化的地方：既要看到其结构化学巨大的基础贡献，也不能回避后期边界感问题。
- 证据不足的部分：不同阶段的研究与公共角色需要继续做时序区分。
