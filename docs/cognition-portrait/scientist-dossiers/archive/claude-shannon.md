# Claude Shannon

## 1. 基本信息

- 姓名：Claude Shannon
- 生卒年：1916 - 2001
- 国别 / 主要活动地区：美国
- 主要学科：信息论
- 代表贡献：奠定信息论与数字电路理论基础，重塑对通信、编码和计算的理解。
- 所处时代：20 世纪中期计算与通信革命期
- 候选池分组：工程、信息、系统与跨界建构
- 当前档案状态：corpus_packaged

## 2. 人物定位（一句话）

- 高度抽象但又保有玩具式实验和工程趣味的整合型建模者。

## 3. 研究气质摘要

- 研究气质：抽象、机智、爱做小装置、低戏剧化
- 问题进入方式：从结构最简化和可量化问题切入
- 推进节奏：抓住关键结构后，给出极高压缩度结果
- 更偏理论 / 实验 / 观察 / 工程：理论 / 工程 / 综合
- 更偏独立 / 合作 / 组织：能在 Bell Labs 体系中工作，但核心创造更像个人抽象突破
- 更偏整合 / 深挖：整合偏强（I）

## 4. 工作方式与方法

- 他通常如何选问题：偏向能把复杂媒介统一成同一数学形式的问题。
- 他如何处理困难或失败：通过抽象化和编码化把问题变成更可操作的结构。
- 他是否擅长长期积累：中高，关键在少数突破性概念。
- 他是否擅长跨界：高，数学、工程、通信、游戏装置之间切换自然。
- 他是否有明显的方法论特征：抽象化、编码、简化、实验小装置验证直觉。

## 5. 表达与公共风格

- 是否属于公众表达型科学家：不是大众型明星，但在技术共同体中极具传奇色彩。
- 公开表达风格：低调、冷幽默、偏简洁。
- 同行 / 外界通常如何形容他的风格：常被描述为“低调的革命者”。
- 是否有鲜明的教学 / 传播能力：在专业圈巨大，在大众层面相对低调。

## 6. 关键转折点

1. MIT 论文把布尔逻辑与开关电路联系起来。
2. Bell Labs 时期形成信息论核心工作。
3. 1948 年经典论文让其成为数字时代关键奠基者。

## 7. 争议、张力与限制

- 主要争议：并非争议中心，但公众认知常远低于其影响力。
- 重要张力：低调个人风格与巨大历史影响并存。
- 不应被过度神化的地方：不能只把他写成“信息论之父”，要看到其玩具式实验与工程感。
- 证据不足的部分：更多第一人称表达材料相对有限。

## 8. 适合作为哪类画像锚点

- 主锚点候选：抽象整合型、冷静建模型、低调高影响型人物

- 更适合支持的维度：开放性、整合、抽象、系统化
- 不太适合支持的维度：高外向舞台型
