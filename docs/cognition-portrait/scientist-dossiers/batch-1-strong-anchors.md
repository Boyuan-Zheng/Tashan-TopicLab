# 第一批强锚点人物（20 位）

这 20 位用于第一轮“强锚点人物”补料。选择原则：

- 公众识别度相对更高
- 研究风格叙事更鲜明
- 能覆盖理论、实验、观察、工程、公共解释、系统组织等不同画像方向
- 相对更容易找到高可信来源

## 名单

| 人物 | 为什么先补 | 当前锚点方向 |
|---|---|---|
| Leonardo da Vinci | 跨学科整合冲动极强、以观察和设计反复回环推进的原型性整合者。 | 跨界整合型、观察-设计联动型、问题网式研究者 |
| Isaac Newton | 高度内向、结构压缩能力极强、长期闭门推演后一次性给出强框架的深挖型理论人物。 | 深挖型理论建构者、基础定律型人物 |
| Charles Darwin | 极其耐心的观察-整合型研究者，通过长期积累和广泛通信把分散事实织成理论。 | 观察-整合型、长期积累型、谨慎发布型人物 |
| Marie Curie | 高度克制、极端耐久、实验与使命感结合很强的深挖型人物。 | 实验耐力型、深挖型、使命驱动型人物 |
| Albert Einstein | 擅长用少数概念性问题重构整体框架的高开放理论整合者。 | 概念重构型、理论整合型、公共科学家型人物 |
| Richard Feynman | 用直观模型、玩耍式试探和高压缩解释推进理解的整合型理论人物。 | 解释型理论人物、好奇驱动型、重构理解型人物 |
| Barbara McClintock | 极端耐心、以长期观察和“贴着材料思考”著称的深挖型实验观察者。 | 长期深挖型、观察-直觉型、晚识别型人物 |
| Rosalind Franklin | 方法论纪律极强、证据门槛极高的精密实验深挖者。 | 证据纪律型、结构实验型、边界清晰型人物 |
| Santiago Ramon y Cajal | 把极高观察力和图像化表达结合在一起的深挖型观察者。 | 观察深挖型、图像化研究型、长期手工积累型人物 |
| Claude Shannon | 高度抽象但又保有玩具式实验和工程趣味的整合型建模者。 | 抽象整合型、冷静建模型、低调高影响型人物 |
| John von Neumann | 极高带宽的跨域整合者，能把抽象数学迅速接到系统设计与组织问题上。 | 系统整合型、跨域高带宽型、组织嵌入型人物 |
| Emmy Noether | 以纯结构、概念统一和理论深掘见长的深挖型抽象人物。 | 抽象深挖型、结构统一型、低外向高影响型人物 |
| Alan Turing | 把极高抽象模型与可实现机器问题连在一起的深挖型桥接者。 | 形式深挖型、理论-机器桥接型、边界定义型人物 |
| Chien-Shiung Wu | 沉稳、精确、靠实验装置与执行纪律取胜的深挖型实验人物。 | 精密实验型、低调高执行型、关键验证型人物 |
| Rachel Carson | 把科学证据、文学表达和公共责任接起来的使命型整合者。 | 公共解释型、使命驱动型、科学写作整合型人物 |
| Grace Hopper | 兼具工程落地、系统组织和教学传播能力的整合型工程人物。 | 工程整合型、组织推广型、教学传播型人物 |
| Herbert A. Simon | 把决策、认知、组织和计算连成一体的系统整合者。 | 系统整合型、认知-组织桥接型、模型化人物 |
| Qian Xuesen | 把国家级工程目标、系统组织和理论方法连成整体的组织型整合者。 | 系统组织型、国家工程型、整合型人物 |
| Vera Rubin | 安静、持久、数据优先的观测型深挖者。 | 观测深挖型、低调高证据型人物 |
| Linus Pauling | 能把理论化学、结构问题和公共科学角色连接起来的高开放整合者。 | 高开放整合型、理论结构型、科学-公共双角色人物 |
