# Definitions

Canonical questionnaire definitions for the scale runtime live here.

Current normalized assets:

- `registry.json`
- `rcss.json`
- `mini-ipip.json`
- `ams-gsr-28.json`

This layer describes:

- scale ids
- question ids
- question text
- dimensions
- score ranges
- reverse-score flags
- definition versions
- derived-metric metadata

It is authoritative for scale meaning, but not responsible for CLI transport or backend persistence.
